================================================================================
                    HERCYCLE EDUCATIONAL PLATFORM
                          USE CASE SUMMARY
================================================================================

PROJECT: Hercycle - Period & Sex Education Platform for Children (Ages 10-18)
TECH STACK: React + Node.js + MongoDB
STATUS: Phases 1-2 Complete, Phase 3+ Planned

================================================================================
                              ACTORS
================================================================================

1. STUDENT/CHILD (Primary Actor)
   - Age Range: 10-18 years
   - Role: Main user of the educational platform
   - Goals: Learn about health, complete modules, earn points

2. SYSTEM (Secondary Actor)
   - Role: Automated processes
   - Functions: Validate tokens, track progress, award points/badges

LEVEL SYSTEM (7 Levels):
  ┌────────┬─────────────────┬────────────────────┐
  │ Level  │ Points Required │ Title              │

  │ 1      │ 0 - 199         │ Beginner           │

  └────────┴─────────────────┴────────────────────┘

BADGE CATEGORIES (5 Types):
  1. MILESTONE: Module completion (1, 3, 6, 9, 12 modules)
  2. PERFECT SCORE: Perfect quiz scores (1, 5, 10 perfect)
  3. STREAK: Daily login streaks (3, 7, 14, 30 days)
  4. POINTS: Total points milestones (500, 1000, 2500, 5000)
  5. SPECIAL: Time-based achievements (Early Bird, Night Owl, etc.)

================================================================================
                       SECURITY & VALIDATION
================================================================================

Age Verification:    Enforced at signup (must be 10-18 years)
Authentication:      JWT tokens with 7-day expiry
Password Security:   Bcrypt hashing with 10 salt rounds
Data Privacy:        Email and gender are optional fields
Input Validation:    All inputs validated on backend
Protected Routes:    JWT required for dashboard, modules, profile
